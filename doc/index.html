Exercise 3

Write some code that accepts an integer and prints the integers from 0 to that input integer in a spiral format.

For example, if I supplied 24 the output would be:

20 21 22 23 24
19  6  7  8  9
18  5  0  1 10
17  4  3  2 11
16 15 14 13 12

Quickstart

open bash and install the dependencides and build the project

yarn install # OR npm install
yarn build # OR npm build

To see the application output, use npm to run the default script with the target value as a parameter

target_value=8
yarn start $target_value # OR npm start $target_value

$ node index.js 8
 6 7 8
 5 0 1
 4 3 2

There are a few unit tests available to validate the application code is working correctly. 

yarn test # OR npm run test
